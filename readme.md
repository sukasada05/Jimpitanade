# 🌾 Jimpitan Bali - Mangsatria

Aplikasi tabungan jimpitan harian berbasis kalender Bali (6 sasih = 210 hari).

## Fitur
- 📱 **PWA** - Bisa install di HP seperti aplikasi native
- 💾 **Setor & Cetak** - Input setoran langsung tersimpan sebagai struk PNG
- 📤 **Kirim WhatsApp** - Struk otomatis terkirim ke nasabah via WA
- ✏️ **Edit/Hapus** - Koreksi transaksi jika salah input
- 📊 **Progress 210 Hari** - Pantau sisa hari setoran
- 🌐 **Offline** - Service Worker cache, tetap jalan tanpa internet

## Teknologi
- **Tailwind CSS** (CDN)
- **Bootstrap Icons** (CDN)
- **html2canvas** (CDN)
- **Google Fonts** - JetBrains Mono
- **Service Worker** - Cache & offline
